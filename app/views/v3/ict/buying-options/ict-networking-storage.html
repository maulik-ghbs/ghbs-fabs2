What it offers

Designed for the education sector this framework offers:

  • on-premise solutions - IT and network infrastructure that are entirely delivered on-premise and includes but not limited to network and storage hardware, back-up, archiving and data protection systems
  • cloud and hybrid cloud solutions - IT infrastructure solutions that are entirely cloud based or a hybrid cloud storage solution, comprising both on-premise and cloud components
  • consultancy services - specifically for ICT consultancy and design services where a solution is yet to be defined, which may fit into either of the solutions above

Benefits

  • save time using the framework rather than completing a full tender process
  • easy and compliant route if you are looking to adopt a cloud solution
  • ability to purchase robust network security solutions

All frameworks also offer

  • compliance with the relevant procurement regulations
  • suppliers pre-checked by the approved framework provider
  • pre-agreed terms and conditions
